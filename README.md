# Roblox Account Checker

Simple tool to check old Roblox accounts.

**WARNING**: Use responsibly. May violate TOS.

## Usage
See roblox.py

Option 1 (Easiest):
PowerShell
python -m pip install requests colorama
Then run the script:
PowerShell
python roblox.py
Option 2:
PowerShell
py -m pip install requests colorama
Then:
PowerShell
py roblox.py
